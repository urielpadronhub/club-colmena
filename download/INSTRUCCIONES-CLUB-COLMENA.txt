═══════════════════════════════════════════════════════════════════
                    ¿CÓMO USAR EL SITIO WEB?
═══════════════════════════════════════════════════════════════════

PASO 1: Descarga el archivo "club-colmena.zip"

PASO 2: Descomprime el archivo (clic derecho → Extraer todo)

PASO 3: Necesitas tener instalado Node.js en tu computadora
        - Descárgalo gratis de: https://nodejs.org
        - Instálalo con las opciones por defecto

PASO 4: Abre una terminal (Símbolo del sistema o PowerShell)
        - Escribe: cd Downloads/my-project (o donde lo descomprimiste)
        - Escribe: npm install
        - Espera a que termine

PASO 5: Ejecuta el servidor
        - Escribe: npm run dev
        - Verás algo como: "Local: http://localhost:3000"

PASO 6: Abre tu navegador (Chrome, Edge, Firefox)
        - Ve a: http://localhost:3000
